# How to take the store screenshots (2 minutes)

Run this from the project root to open each mockup at the exact right size:

```
node store-assets/open-screenshots.mjs
```

Or manually:

## Screenshot 1 — Scanning screen (1280×800)
1. Open `store-assets/screenshot-1-scanning.html` in Chrome
2. Press F12 → click the phone/tablet icon (Toggle device toolbar)
3. Set dimensions to **1280 × 800** → press Enter
4. Close DevTools
5. Press **Ctrl+Shift+P** → type "screenshot" → click **Capture screenshot**
6. Save as `store-assets/screenshot-1-scanning.png`

## Screenshot 2 — Results dashboard (1280×800)
1. Open `store-assets/screenshot-2-results.html` in Chrome
2. Same steps as above → save as `store-assets/screenshot-2-results.png`

## Promo tile (440×280)
1. Open `store-assets/promo-tile.html` in Chrome
2. Set device toolbar to **440 × 280**
3. Capture screenshot → save as `store-assets/promo-tile.png`

---
Upload these 3 files to the Chrome Web Store listing.
